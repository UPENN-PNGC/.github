# .github

This repository contains default community health files and templates for the UPENN-PNGC organization. These files are automatically used across all repositories in the organization unless overridden by repository-specific files.

## Contents

### Community Health Files

- **[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)** - Our community code of conduct based on the Contributor Covenant
- **[CONTRIBUTING.md](CONTRIBUTING.md)** - Guidelines for contributing to our projects
- **[SECURITY.md](SECURITY.md)** - Security policy and vulnerability reporting procedures
- **[SUPPORT.md](SUPPORT.md)** - How to get help and support

### Templates

- **Issue Templates** (`.github/ISSUE_TEMPLATE/`)
  - Bug report template
  - Feature request template
  - Configuration for issue creation

- **Pull Request Template** (`.github/PULL_REQUEST_TEMPLATE.md`)
  - Standard template for pull requests across all repositories

### Organization Profile

- **[profile/README.md](profile/README.md)** - Organization profile displayed on the UPENN-PNGC GitHub page

## How It Works

GitHub automatically uses files from this `.github` repository as defaults for all repositories in the UPENN-PNGC organization. Individual repositories can override these defaults by including their own versions of these files.

## Customization

To customize these files for a specific repository:

1. Copy the desired file(s) from this repository
2. Place them in the target repository at the same relative path
3. Modify as needed for that repository's specific requirements

## Maintenance

These files should be reviewed and updated periodically to ensure they remain relevant and useful for all organization repositories.
