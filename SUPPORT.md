# Support

Thank you for using UPENN-PNGC projects! We're here to help.

## Getting Help

PNGC is an academic organization and does not directly manage many affiliated GitHub repositories. For project or application specific questions or support, please contact the maintainer or support contact listed in the relevant repository.

### Documentation

First, check the documentation:
- Repository README files
- Project wikis
- API documentation
- Example code and tutorials

### Common Issues

Before seeking help, check if your issue is addressed in:

- The README or documentation section of the relevant repository
- The Issues tab of the repository for open or closed issues with similar problems

## Ways to Get Support

### 1. GitHub Issues

For bugs, feature requests, or technical problems:
- Search existing issues first
- Create a new issue in the relevant repository
- Provide detailed information:
  - What you're trying to do
  - What you expected to happen
  - What actually happened
  - Steps to reproduce
  - Environment details

### 2. Discussions

For questions, ideas, and general discussion:
- Use GitHub Discussions (if enabled for the repository)
- Share your use case
- Ask for clarification
- Discuss best practices

### 3. Community

- Collaborate with other users
- Share your experiences
- Help others when you can

## What We Support

✅ **We can help with:**
- Bugs in the software
- Feature requests
- Documentation clarifications
- Installation and setup issues
- Usage questions

❌ **We cannot help with:**
- Custom implementations
- Third-party tools
- General programming help
- Unrelated technical issues

## Response Times

Please note:
- This is an academic/research organization
- Response times may vary
- Complex issues may take longer to address
- We prioritize critical bugs and security issues

## Contributing

If you've found a solution to a common problem:
- Consider contributing to the documentation
- Submit a pull request with fixes
- Help others in discussions and issues

## Code of Conduct

All interactions must follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Research Collaboration

For research collaborations or academic inquiries, please refer to the specific repository or organization contact information.
